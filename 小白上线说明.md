## 整页纯白 / 一直转圈？

很常见：**国内网络访问 Vercel（美国）有时拉不动 JavaScript**。请依次试：换 **4G/5G**、关掉**广告拦截**、换 **Chrome** 或 **微信内置浏览器**、过一段时间再开。页顶灰色条也会提示这一点。

## 还必须你做的一件事（不然分数可能存不住）

现在网站在云端**没有接数据库**，多几个人打分、或过一阵子，**数据可能丢或乱**。

请花 10 分钟接一个免费的 **Supabase**（像网盘一样帮你存分数）：

1. 打开 [https://supabase.com](https://supabase.com) → 用邮箱注册 / 登录（免费）。
2. **New project** 新建一个项目，选一个离中国近的区（例如 **Singapore**），设个数据库密码，等它变绿（一两分钟）。
3. 左侧 **SQL Editor** → **New query**，把本仓库里的文件 **`supabase/schema.sql` 全部复制进去** → 点 **Run**（整份一次执行）。
4. 左侧 **Project Settings**（齿轮）→ **API**，复制三个东西：
   - **Project URL**
   - **anon public** key
   - **service_role** key（点眼睛才看得见，不要发给任何人）
5. 打开 [vercel.com](https://vercel.com) → 你的项目 **kexuesuanming** → **Settings** → **Environment Variables**，新增三条（名字必须一模一样，选 **Production**）：

   | Name | 填什么 |
   |------|--------|
   | `NEXT_PUBLIC_SUPABASE_URL` | 第 4 步的 Project URL |
   | `NEXT_PUBLIC_SUPABASE_ANON_KEY` | 第 4 步的 anon public |
   | `SUPABASE_SERVICE_ROLE_KEY` | 第 4 步的 service_role |

6. 回到 **Deployments** 面板，对最新一条点 **⋯** → **Redeploy**。

做完以后，打分数据就会稳稳存在数据库里。
